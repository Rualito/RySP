============
Introduction
============

This is the documentation of the Rydberg Simulation Platform :py:mod:`rsp` package. Its purpose is to accuratelly simulate the physics of Rydberg atoms, while also having high versatility when implementing complex quantum circuits. 


Instalation
===========

The package can be installed using pip:

``pip install git://https://gitlab.tue.nl/20235021/rsp``

(see :ref:`install`)
